Dette er min fullstendige innlevering av en notatapplikasjon. Her kan brukeren skrive inn notater med tittel, innhold og kategori, og todo-oppgaver. Notater kan festes og todos kan markeres som utført, i tillegg til at begge deler kan redigeres og slettes. Todos kan også filtreres etter hvilke som er utført og hvilke som fremdeles er aktive. Jeg har også lagt til muligheten til å velge mellom lys og mørk modus på siden. Alt kjøres fra terminal, så det eneste brukeren må gjøre er skrive "python3 app.py" i terminal også startes applikasjonen. Dataene som puttes inn lagres i en SQLite database. Notater bruker "id", "tittel", "content", "pinned" og "created" for å lagre datene. Det betyr at der er mulighet for å legge inn filter for notater også for å gjøre det lettere for brukeren å finne frem til notatene sine, men jeg har ikke tid til å legge til denne funksjonen nå. Todos bruker "id", "task" og "done", derfor har den en mulighet for å filtrere etter hvilke som er utført og ikke.
