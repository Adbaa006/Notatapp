Dette er min fullstendige innlevering av en notatapplikasjon. Her kan brukeren skrive inn notater med tittel, innhold og kategori, og todo-oppgaver. Notater kan festes og todos kan markeres som utført, i tillegg til at begge deler kan redigeres og slettes. Todos kan også filtreres etter hvilke som er utført og hvilke som fremdeles er aktive. Jeg har også lagt til muligheten til å velge mellom lys og mørk modus på siden. Alt kjøres fra terminal, så det eneste brukeren må gjøre er skrive "python3 app.py" i terminal også startes applikasjonen.
